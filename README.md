## 🤖 About ROOAR

> **ROOAR-BOT** is a powerful and customizable WhatsApp Multi-Device bot using [Baileys](https://github.com/whiskeysockets/Baileys).
> It includes AI, media tools, group features, moderation, sticker generation, and more.

> ⚡ Originally inspired by [Hisoka-Morou](https://github.com/DikaArdnt/Hisoka-Morou)
> 🧥 Forked through [RAVEN-BOT](https://github.com/HunterNick2/RAVEN-BOT)
> 💫 Now re-engineered and enhanced under **Genesis | Tace Tygre** branding as **ROOAR-BOT**

---

## 🚀 Features

✅ WhatsApp MD (Baileys)
✅ Gemini, ChatGPT, Bard AI
✅ YouTube, TikTok, IG Downloader
✅ Group Tools (kick/promote/mute)
✅ Sticker Maker + Tools
✅ Welcome/Goodbye
✅ Fake Commands
✅ Custom Replies
✅ Moderation Tools
✅ API integrations
✅ Obfuscated Deployment
✅ MongoDB / QuickMongo support
✅ Express / Web UI Ready

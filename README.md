## ⚙️ DESCRIPTION

**ROOAR-BOT** is a WhatsApp Multi-Device bot built with `Node.js` using the powerful [Baileys](https://github.com/whiskeysockets/Baileys) library.  
It includes AI, media tools, group moderation, stickers, downloader, and more.

---

## 🚀 FEATURES

✅ WhatsApp Multi-Device  
✅ ChatGPT, Gemini, Bard AI Support  
✅ YouTube, Instagram, TikTok Downloader  
✅ Group Tools (kick, promote, demote)  
✅ Fun tools (quotes, fake replies, memes)  
✅ Anime, Image Search, OCR  
✅ Sticker Maker  
✅ Welcome/Goodbye  
✅ Anti-Delete & Anti-ViewOnce  
✅ And more...
